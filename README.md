V-Pet is a browser-based virtual pet web application inspired by games like Pou. The project was developed as a final assignment for a web module by a team of three developers. 

[Mark](https://github.com/Markkndr)
[Doma](https://github.com/makkdomonkos)

Built with React + Vite (JavaScript), the app allows users to take care of their own virtual pet, interact with it, and track its status in real time. The application uses Firebase as its backend for data storage and synchronization.

🚀 Features
Create and manage your own virtual pet
Real-time state updates using Firebase
Interactive UI with responsive design
Simple and fun gameplay experience
🛠 Tech Stack
React
Vite
JavaScript
Firebase
